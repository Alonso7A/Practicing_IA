Capítulo 02

Conceptos clave que debes conocer

El vocabulario esencial de Git y GitHub explicado con palabras simples. Es la base sobre la que se apoyan todos los capítulos siguientes.

Antes de empezar a teclear comandos, familiarízate con este vocabulario. Lo encontrarás en cada tutorial, error y conversación sobre Git. No hace falta memorizarlo de golpe: vuelve aquí cuando lo necesites.

Tarjetas de vocabulario

Repositorio (repo)

La carpeta de tu proyecto, junto con todo el historial de cambios. Puede ser local (en tu PC) o remoto (en GitHub).

Commit

Una "fotografía" de tus archivos en un momento dado. Cada commit tiene un mensaje, un autor y un identificador único (hash).

Branch (rama)

Una línea paralela de desarrollo. Te permite trabajar en una funcionalidad sin afectar el código principal.

Merge

Fusionar los cambios de una rama dentro de otra. Es cómo se incorpora tu trabajo a la rama principal.

Push / Pull

push sube tus commits al servidor remoto. pull descarga lo que otros subieron.

Clone

Descargar un repositorio remoto completo a tu máquina para empezar a trabajar en él.

Staging area

Zona intermedia donde colocas los archivos que quieres incluir en tu próximo commit. Se llena con git add.

Pull Request (PR)

Solicitud para fusionar una rama en otra. Es donde ocurre la revisión de código entre colaboradores.

Remote / Origin

Remote es la dirección de un repo en internet. Origin es el alias por defecto del remoto principal.

Main / Master

La rama principal del proyecto. Hoy se prefiere main; los repos antiguos usan master.

Issue

Un "ticket" o tarea/bug abierto en el proyecto. Sirve para discutir y dar seguimiento a problemas o ideas.

Fork

Copia personal de un repo ajeno. Te permite modificarlo libremente y luego proponer tus cambios al original.

Resumen visual

Si tuvieras que recordar solo 3 ideas

1. Un commit es una foto. El historial es un álbum de fotos.

2. Una rama es una copia paralela donde experimentas sin romper lo principal.

3. Push sube, pull baja. El resto son matices de estas dos acciones.
